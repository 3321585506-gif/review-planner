# 今天你鼠了吗？

这是一个纯静态复习计划网站，可以直接部署到 GitHub Pages。

## 上传到 GitHub Pages

1. 在 GitHub 新建公开仓库，例如 `review-planner`。
2. 上传本文件夹里的所有内容：
   - `index.html`
   - `styles.css`
   - `app.js`
   - `assets/logo.gif`
   - `README.md`
3. 打开仓库的 `Settings` -> `Pages`。
4. 选择 `Deploy from a branch`，分支选 `main`，目录选 `/root`。
5. 保存后等待 1-3 分钟，GitHub 会生成访问链接。

## 注意

勾选完成、拖动任务、延后和提前等状态保存在当前浏览器本地。平板和电脑之间不会自动同步。
